# doExcel
********CSV文本数据处理程序--比对数据前后一致********  
使用说明： 
1.执行程序前将undoSheet.csv放在程序的同一目录 
2.按照格式填充ABCD四列，分别填充上次统计用户名，上次统计分数，本次统计用户名，本次统计分数; 
3.用户名中末尾不能包含空格，比如“小军哥  ”和“小军哥”会判定为不同用户 4.最长限定400个用户数据，超出无法执行。
